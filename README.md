# Titanic survivors prediction using Flask API

## usage
clone the repository
```bash
$ git clone https://github.com/kandeldeepak46/docker_REST_api.git
$ cd docker_REST_api
$ pip install -r requirements.txt
$ python api.py

```

after running the server you can train the model by navigating
```
localhost://train
```
in you browser


In another bash [for testing the api]

```bash
$ python example_http_request.py
```
